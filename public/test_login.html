🔐 Test Login Button

راهنما

  1. 1. این صفحه را باز نگه دارید
  2. 2. در تب دیگر به https://www.zala.ir بروید
  3. 3. Console را باز کنید (F12)
  4. 4. به صفحه login بروید (اگر login هستید، logout کنید)
  5. 5. این دستورات را در Console تایپ کنید:

دستورات Console

1. بررسی وجود Form:

document.getElementById('loginForm')

باید یک HTMLFormElement نشان دهد

2. بررسی وجود Button:

document.getElementById('loginBtn')

باید یک HTMLButtonElement نشان دهد

3. بررسی Event Listeners:

const form = document.getElementById('loginForm');
console.log('Form:', form);
console.log('Form onsubmit:', form.onsubmit);
console.log('Form listeners:', getEventListeners(form));

باید event listener submit را نشان دهد

4. تست دستی Submit:

document.getElementById('username').value = 'testuser';
document.getElementById('password').value = 'Test@123';
document.getElementById('loginForm').submit();

⚠️ این refresh می‌شود (نرمال است)

5. تست با dispatchEvent:

document.getElementById('username').value = 'testuser';
document.getElementById('password').value = 'Test@123';
const form = document.getElementById('loginForm');
form.dispatchEvent(new Event('submit', {bubbles: true, cancelable: true}));

این handleLogin را فراخوانی می‌کند

6. تست دستی handleLogin:

document.getElementById('username').value = 'testuser';
document.getElementById('password').value = 'Test@123';
app.handleLogin({preventDefault: () => {}});

مستقیماً handleLogin را صدا می‌زند

7. بررسی TitanApp:

console.log('window.app:', window.app);
console.log('window.titanApp:', window.titanApp);
console.log('Has handleLogin:', typeof (window.app?.handleLogin || window.titanApp?.handleLogin));

بررسی instance اپلیکیشن

💡 نکات:

  • • اگر form یا button null است → صفحه login نمایش داده نشده
  • • اگر event listener نیست → setupEventListeners اجرا نشده
  • • اگر handleLogin کار نمی‌کند → مشکل در app instance است
  • • اگر submit می‌شود اما login نمی‌شود → مشکل در API است

⚠️ اگر دکمه کار نمی‌کند:

  • 1. ✅ Form و Button را بررسی کنید (دستورات 1 و 2)
  • 2. ✅ Event listeners را بررسی کنید (دستور 3)
  • 3. ✅ تست دستی کنید (دستورات 4، 5، 6)
  • 4. ✅ خطاهای Console را بخوانید
  • 5. ✅ Screenshot بگیرید و نشان دهید
